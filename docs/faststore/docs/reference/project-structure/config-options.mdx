The `discovery.config.js` file is the central configuration file for managing store settings.
It allows you, for example, to set SEO settings, define the store ID, select a theme, and set up custom Cypress tests. Check out the example for this file: [`discovery.config.js` sample code](https://github.com/vtex-sites/starter.store/blob/main/discovery.config.js).
