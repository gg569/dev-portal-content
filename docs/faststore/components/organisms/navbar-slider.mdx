As part of the [Navbar](https://developers.vtex.com/docs/guides/faststore/organisms-navbar), this component is its **mobile version** used to list the navigation links (through the [NavbarLinks](https://developers.vtex.com/docs/guides/faststore/molecules-navbar-links)).

This component is part of the [Navigation](https://developers.vtex.com/docs/guides/faststore/feature-navigation) feature and must be used in the [Navbar](https://developers.vtex.com/docs/guides/faststore/organisms-navbar) context.

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import {
  NavbarSlider,
  NavbarSliderHeader,
  NavbarSliderContent,
  NavbarSliderFooter,
} from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/NavbarSlider/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.

## Usage

> ⚠️ This component is intended to be used only in the **mobile version**, but it can be adapted to a desktop version by customizing the component styles.

<Tabs items={['Example', 'Code']} defaultIndex="0">
  <Tab>
    <OverviewSection containerStyle={{ display: 'block' }}>
      <UIProvider>
        <NavbarSliderUsage />
      </UIProvider>
    </OverviewSection>
  </Tab>
  <Tab>
  
```tsx
  const { closeNavbar } = useUI()
  const { fade, fadeOut } = useFadeEffect()

  return (
    <NavbarSlider
      fade={fade}
      onDismiss={fadeOut}
      onTransitionEnd={() => fade === 'out' && closeNavbar()}
    >
      <NavbarSliderHeader onClose={fadeOut}>
        <IconButton
          onClick={fadeOut}
          aria-label="Logo"
          icon={<Icon name="Storefront" width={32} height={32} />}
        />
      </NavbarSliderHeader>
      <NavbarSliderContent>
        <NavbarLinksUsage />
      </NavbarSliderContent>
      <NavbarSliderFooter>
        <Button onClick={fadeOut}>Sign In</Button>
      </NavbarSliderFooter>
    </NavbarSlider>
  )
```

  </Tab>
</Tabs>
