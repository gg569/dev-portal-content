The `SearchInput` is responsible for receiving, searching, and displaying suggestions. This component is part of the [Search](https://developers.vtex.com/docs/guides/faststore/feature-search) feature.

## Import

Import the component from [@faststore/ui](https://developers.vtex.com/docs/guides/faststore/components-index)

```tsx
import { SearchInput } from '@faststore/ui'
```

### Import Styles into your FastStore project

To apply the styles of this component in your FastStore project, import the following into your stylesheet:

```scss
@import '@faststore/ui/src/components/organisms/SearchInput/styles.scss';
```

Follow the instructions in the [Importing FastStore UI component styles](https://developers.vtex.com/docs/guides/faststore/using-themes-importing-ui-components-styles) tutorial.

      <SearchInputField
        placeholder="Search everything at the store"
        onFocus={() => setSearchDropdownVisible(true)}
        onSubmit={() => setSearchDropdownVisible(false)}
        onChange={(e) => onChangeValue(e)}
        buttonProps={{
          onClick: onClickButton,
